## 如何使用

### 运行自动化测试
1. 安装Python依赖：
```bash
cd automation_tests
pip install -r requirements.txt
```

2. 运行测试：
```bash
pytest test_login_functional.py -v
```

3. 生成HTML报告：
```bash
pytest test_login_functional.py --html=report.html
```
